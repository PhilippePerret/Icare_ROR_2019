Description de l'action-watcher
===============================

Action-watcher me permettant de télécharger les commentaires.

L'icarien(ne) est averti(e) et peut rejoindre son bureau pour les charger.

Dans l'attente, un formulaire d'upload des documents est présenté à l'administrateur et une notice indiquant la date de remise des commentaires (avec le délai restant) est indiqué sur le bureau de l'icarien.
